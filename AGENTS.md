# Project Instructions

- After any UI change, reopen the Electron app automatically so the latest interface is visible without the user asking again.
- Match the existing calm visual language across the app.
- Do not use green pulsing dots, status dots, dotted lines, or similar animated status decorations in the UI.
- Prefer simple text labels or badges over ornamental status indicators.
- Do not add tutorial-style helper banners, onboarding callouts, or explanatory empty-state coaching copy in the UI unless the user explicitly asks for it.
